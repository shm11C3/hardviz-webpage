# hardviz-webpage

<https://hardviz.com/>

Official website for HardwareVisualizer, a hardware visualization tool.

## Tech Stack

- Astro
- React
- Tailwind CSS
- TypeScript
- Playwright (E2E)

## Setup

```bash
pnpm install --frozen-lockfile
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Preview

```bash
pnpm preview
```

## Test

```bash
pnpm test
```

## Project Structure

```txt
src/
  components/   UI components
  pages/        routes (Astro)
  layouts/      layouts
  i18n/         translations
  styles/       global CSS
tests/          Playwright E2E
```
